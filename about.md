## Hello, I'm {{ site.author.name | default: "the author" }}.

This blog is built with the **Glass Theme** — a glassmorphism Jekyll theme featuring:

- Frosted glass UI cards with backdrop-filter blur
- Animated aurora / orb backgrounds
- A responsive two-column post grid with a featured hero post
- Auto reading progress bar
- Copy-to-clipboard code blocks
- Active TOC highlighting on scroll
- Zero dependencies — pure HTML, CSS, and vanilla JS

### About Glass Theme

Glass Theme is crafted by [csswitch](https://github.com/csswitch) — a collection of distinctive Jekyll themes for developers who care about aesthetics.

**Features:**
- 🔮 Glassmorphism design — frosted glass cards, orb backgrounds
- 🎨 Configurable accent color (any hue via `_config.yml`)
- 📱 Fully responsive — mobile-first
- ⚡ No jQuery, no frameworks — vanilla JS only
- 🐙 GitHub Pages compatible out of the box
- ♿ Accessible — keyboard nav, ARIA labels, focus rings
- 🔍 SEO optimized with jekyll-seo-tag
- 📡 RSS feed with jekyll-feed
